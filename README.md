# About the project

A Sorting algorithm visualizer to get a better understanding of how different sorting algorithms works.
Sorting Algorithms visualized in this project are as follows:

1. Bubble Sort.
2. Selection Sort.
3. Insertion Sort.
4. Quick Sort.
5. MergeSort.

# Run Project on your system

1. Git Clone the project. (git clone https://github.com/nitin-meta/sorting-visualizer.git)
2. cd into the project folder. (cd sorting-vizualizer)
3. Install the dependencies using (npm install)
4. Run the project. (npm start)
